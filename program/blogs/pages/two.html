Переменные и строгий режим

Переменные — это хранилища информации

Виды переменных Описание Пример
let Значение переменной может быть перезаписано let number = 5
const Значение переменной не может быть перезаписано (ну почти :D ) const leftBorderWidth = 5
var Проблема var в том что она сущестует еще то того как она была объявлена в коде и видна она везде var Name = 'Alex'
Глобальные Переменные, объявленные вне всех функций и блоков, имеют глобальную область видимости и доступны из любой части кода. let modalBlock = modal;
Локальные Переменные, объявленные внутри функции или блока, имеют локальную область видимости и доступны только внутри этой функции или блока. function test() { let test = 'test'; }
Объектные Переменные могут хранить объекты, массивы и функции, что позволяет создавать сложные структуры данных. let obj = { name: 'Modal', classToggle: active };

  • Имя переменной должно начинаться с буквы (a-z, A-Z), знака доллара ($) или символа подчеркивания (_).
  • Используйте стиль camelCase для многосоставных имен
  • Имена переменных чувствительны к регистру
  • Прямых констант в js не бывает
  • Нельзя начинать имя переменной с цифры.
  • Нельзя использовать зарезервированные слова JavaScript

"use strict"; — это директива в JavaScript, которая активирует строгий режим, это значит что теперь недопускается работы некоторых нечтоностей котоые были в старых стандартах javascript
