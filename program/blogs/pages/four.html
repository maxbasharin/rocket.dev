Классификация типов данных в JavaScript

Существует 8 типов данных

Делятся на простые типы (примитивы) объекты (комплексные)

Простые типы данных

Тип данных Описание Пример
Числа Представляют числовые значения, как целые, так и дробные. let age = 30;
Строки Последовательности символов, заключенные в одинарные или двойные кавычки. let name = "John";
Логический тип (Boolean) Принимает два значения: true или false. let isActive = true;
Undefined Переменная, которая была объявлена, но не инициализирована. let user; // user имеет значение undefined
Null Намеренное отсутствие значения. let selectedItem = null;
Symbol Уникальный и неизменяемый примитивный тип, используемый как идентификатор. const uniqueId = Symbol('id');
BigInt Позволяет работать с целыми числами произвольной длины. const bigNumber = 1234567890123456789012345678901234567890n;


Специальные объекты

Тип объекта Описание Пример
Массивы (Array) Упорядоченные коллекции значений. const fruits = ['apple', 'banana'];
Функции (Function) Объекты, которые могут быть вызваны для выполнения кода. function greet() { console.log("Hello!"); }
Объекты Даты Используются для работы с датами и временем. const currentDate = new Date();
Регулярные выражения Шаблоны для поиска и манипуляции строками. const regex = /abc/;
Ошибки (Error) Объекты, представляющие ошибки, возникающие во время выполнения. const error = new Error("Something went wrong!");


Специальные объекты

Тип объекта Описание Пример
Обычные объекты Наборы пар "ключ-значение", которые могут содержать данные и функции. const user = { name: "John", age: 30 };
