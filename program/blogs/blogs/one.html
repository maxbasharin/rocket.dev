Что такое JS и как его подключить к странице

JavaScript — это универсальный язык программирования, который в первую очередь используется для создания интерактивных элементов на веб-страницах

Способ подключения Описание
Внутренний скрипт JavaScript код помещается внутри тега <script> alert(1); </script> в HTML-файле.
Внутренний скрипт <button onclick="">меню</button>
Внешний скрипт Внешний файлик с расширением .js <script src="script.js"></script>
